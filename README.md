# Bolo
Multiplayer tank strategy game from the 80s.

## Debug Run
`./gradlew :jvm-app:run`

## Dist
`./gradlew :jvm-app:assembleDist`

## Configure with bolo.properties
* MAP
* HTTP_PORT
* HTTPS_PORT
* KEYSTORE_PATH
* KEY_ALIAS
* KEY_STORE_PASSWORD
* PRIVATE_KEY_PASSWORD
